# 图片生成 / 图像编辑（Images Generations）

SiliconFlow 图片生成接口，支持文生图、图生图和部分模型的多图编辑。接口会返回生成图片的 URL，生产环境建议及时下载并存储到自己的文件系统或对象存储。

官方文档：https://api-docs.siliconflow.cn/docs/api/images-generations-post

## 接入信息

| 项 | 值 |
| :--- | :--- |
| Endpoint | `POST https://api.siliconflow.cn/v1/images/generations` |
| 鉴权 | `Authorization: Bearer <SILICONFLOW_API_KEY>` |
| `Content-Type` | `application/json` |
| 文生图示例模型 | `Kwai-Kolors/Kolors` |
| 图像编辑示例模型 | `Qwen/Qwen-Image-Edit-2509` |

模型、尺寸和参数支持范围会随平台调整，实际以 SiliconFlow 控制台 Models 页面和官方接口文档为准。

## 请求体（常用字段）

| 字段 | 类型 | 场景 | 说明 |
| :--- | :--- | :--- | :--- |
| `model` | string | 文生图 / 图生图 | 必填，模型名称 |
| `prompt` | string | 文生图 / 图生图 | 必填，画面描述 |
| `negative_prompt` | string | 文生图 / 图生图 | 可选，负向提示词 |
| `image_size` | string | 文生图 / 部分图生图 | 可选或按模型要求必填，格式如 `1024x1024`、`1664x928` |
| `batch_size` | integer | `Kwai-Kolors/Kolors` | 可选，输出图片数量，范围 `1` 到 `4` |
| `seed` | integer | 文生图 / 图生图 | 可选，随机种子 |
| `num_inference_steps` | integer | 文生图 / 图生图 | 可选，推理步数，默认 `20`，范围 `1` 到 `100` |
| `guidance_scale` | number | `Kwai-Kolors/Kolors` | 可选，提示词贴合度，默认 `7.5` |
| `cfg` | number | Qwen Image 系列 | 可选，提示词贴合度，范围 `0.1` 到 `20` |
| `image` | string | 图生图 / 图像编辑 | 可选，参考图，支持 URL 或 `data:image/png;base64,...` |
| `image2` | string | `Qwen/Qwen-Image-Edit-2509` | 可选，第二张参考图 |
| `image3` | string | `Qwen/Qwen-Image-Edit-2509` | 可选，第三张参考图 |

## 文生图

### cURL

```bash
curl --request POST \
  --url https://api.siliconflow.cn/v1/images/generations \
  --header "Authorization: Bearer ${SILICONFLOW_API_KEY}" \
  --header "Content-Type: application/json" \
  --data '{
    "model": "Kwai-Kolors/Kolors",
    "prompt": "A cat on a windowsill, soft daylight, photorealistic",
    "image_size": "1024x1024",
    "batch_size": 1,
    "num_inference_steps": 20,
    "guidance_scale": 7.5
  }'
```

### Python

```python
import os
import requests

url = "https://api.siliconflow.cn/v1/images/generations"
headers = {
    "Authorization": f"Bearer {os.environ['SILICONFLOW_API_KEY']}",
    "Content-Type": "application/json",
}
payload = {
    "model": "Kwai-Kolors/Kolors",
    "prompt": "A cat on a windowsill, soft daylight, photorealistic",
    "image_size": "1024x1024",
    "batch_size": 1,
    "num_inference_steps": 20,
    "guidance_scale": 7.5,
}

response = requests.post(url, headers=headers, json=payload, timeout=120)
response.raise_for_status()
data = response.json()

for item in data["images"]:
    print(item["url"])
```

## 图生图 / 单图参考

```python
import os
import requests

url = "https://api.siliconflow.cn/v1/images/generations"
headers = {
    "Authorization": f"Bearer {os.environ['SILICONFLOW_API_KEY']}",
    "Content-Type": "application/json",
}
payload = {
    "model": "Kwai-Kolors/Kolors",
    "prompt": "把参考图改成黄昏海边灯塔场景，保持主体构图",
    "image_size": "1024x1024",
    "batch_size": 1,
    "num_inference_steps": 20,
    "guidance_scale": 7.5,
    "image": "https://example.com/reference.png",
}

response = requests.post(url, headers=headers, json=payload, timeout=120)
response.raise_for_status()
print(response.json())
```

## 多图编辑（Qwen Image Edit）

```python
import os
import requests

url = "https://api.siliconflow.cn/v1/images/generations"
headers = {
    "Authorization": f"Bearer {os.environ['SILICONFLOW_API_KEY']}",
    "Content-Type": "application/json",
}
payload = {
    "model": "Qwen/Qwen-Image-Edit-2509",
    "prompt": "把三张参考图中的主体组合成一张产品宣传图，背景保持干净",
    "num_inference_steps": 20,
    "cfg": 4,
    "image": "https://example.com/reference-1.png",
    "image2": "https://example.com/reference-2.png",
    "image3": "https://example.com/reference-3.png",
}

response = requests.post(url, headers=headers, json=payload, timeout=120)
response.raise_for_status()
print(response.json())
```

## 响应与调试

成功响应示例：

```json
{
  "images": [
    {
      "url": "https://..."
    }
  ],
  "timings": {
    "inference": 0.1
  },
  "seed": 0
}
```

排查问题时建议记录响应头中的 `x-siliconcloud-trace-id`、模型名、图片数量、参考图数量、耗时、成功 / 失败结果和错误摘要。不要记录 API Key、完整 prompt、完整用户隐私文本或私有图片 URL。
