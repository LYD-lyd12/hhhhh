==========================================
 TeleToken Router - AI 模型聚合平台
 本地运行，2 步启动
==========================================

系统要求
------------------------------------------
- Windows 10/11 (64 位)
- Node.js (https://nodejs.org，推荐 LTS 版本)

快速开始（2 步）
------------------------------------------

  第 1 步：双击 "安装依赖.bat"
     -> 自动安装前端和后端所需的所有依赖包
     -> 只需运行一次，以后不用再装

  第 2 步：双击 "start-all.bat"
     -> 自动启动前后端服务
     -> 浏览器打开 http://localhost:3000 即可使用

默认账号
------------------------------------------
  管理员: admin@teletoken.io / admin123
  测试用户: test@teletoken.io / test123
  API Key: admin-api-key-123
  推荐模型: deepseek-chat

文件说明
------------------------------------------
  安装依赖.bat     - 首次使用前运行，安装依赖
  start-all.bat    - 启动前后端（本地运行）
  start.bat        - 仅启动前端
  停止服务.bat     - 关闭所有服务

注意事项
------------------------------------------
  - 电脑重启后需重新双击 "start-all.bat"
  - 本项目为本地运行，无需内网穿透
  - API 调用配置：Base URL = http://localhost:8080/api/v1，Key = admin-api-key-123

问题排查
------------------------------------------
  Q: "安装依赖.bat"报错？
  A: 检查是否装了 Node.js，网络是否正常

  Q: 页面打不开？
  A: 确认 start-all.bat 已运行，等几秒后刷新 http://localhost:3000
